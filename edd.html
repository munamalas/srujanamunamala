    ABOUT

    The goal is to create a responsive web design for a government organization of our choice. We decided to redesign the website for the Employment Development Department (EDD).  The EDD focuses on making sure that citizens in California get their employment benefits should anything occur. I was focusing on making sure that the users can optimially search for their new dream job.

    FINAL SCREENS

    Cal Job Portal-Desktop Version

    These are the final screens for the Cal Jobs Portal that I created. The user would have two options. Either he or she can enter the job that they want using the search bar at the top of the page. They could also click on the corresponding category of interest. This would lead to the available job listings as shown. For the search bar, I wrote the corresponding text entry requested above the box. As humans, we tend to have limited attention spans. Also, some individuals have short term memory loss. This makes it harder for the individual to remember what he or she was supposed to type. I created a toggle that filters the results between Job Openings and Top Companies. Job Openings show the jobs that are currently seeking future employees. Top Companies filters the results based on ratings. It will also show how many positions are available and allow the user to see what the job post is and decide whether they want to apply for the position or not.

    NAVIGATION SCREENS

    This is the new home screen and navigation menu that I designed for the website.   If you remember the original layout, you'd remember that the whole page was covered with information.  This made it very confusing for the user to get from point A to B without any issues.  So, I took only the essential links from each section and made the menu accordingly.  The secondary information went towards the bottom. I believed that the main goal of the website was to be as helpful.  This is why I created a contingency plan.  I displayed the top 3 links on the home screen.  This is only in case a user does not want to go directly to the page through the menu.

    STYLE GUIDE

    This is the style guide that we developed for this project.  The main colors we used were blue, yellow, and white.  We used blue to create a calm feeling  for the website.  We used shades of yellow and orange so that we could maintain the California theme throughout the website.  The white was a text color for the headings. The fonts we used were Raleway, Helvitica, and Arial.  Raleway is generally used as a heading which is why we decided to use that font.  This was mainly used for the main headings of the page (e.g. header,  and title of the page).  Helvetica Neuve is generally used as text font.  We chose this font because we believed that it was the font that was legible.  Legibility is an important factor to consider when re-designing a website of a government organization.  This font was used for sub-headings (e.g. main category cards).  Arial was another font we used for text. This was mainly used for paragraph text within the cards.
